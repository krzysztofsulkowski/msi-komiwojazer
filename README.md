# MSI Komiwojażer

## Opis projektu

Projekt został wykonany w ramach przedmiotu **Metody Sztucznej Inteligencji**.

Celem aplikacji jest stworzenie prostego systemu wsparcia logistycznego, który wyznacza możliwie krótką trasę dla kuriera odwiedzającego wiele punktów i wracającego na końcu do punktu startowego. Program ma minimalizować dystans przejazdu, a jednocześnie działać sprawnie obliczeniowo.

Aplikacja została przygotowana jako **wersja desktopowa w Pythonie**.

# Instrukcja uruchomienia

## Wymaganie wstępne
- mieć Pythona 3.12

## Uruchomienie programu

1. Przejdź do terminalu aplikacji

2. Sklonuj repozytorium

`git clone https://github.com/krzysztofsulkowski/msi-komiwojazer.git`

3. Przejdź do folderu z projektem

`cd msi-komiwojazer`

4.Utwórz i aktywuj środowisko wirtualne

`python -m venv .venv`

`.venv\Scripts\activate`

5. Zainstaluj zależności

`pip install -r requirements.txt`

6. Uruchom aplikację

`python main.py`
